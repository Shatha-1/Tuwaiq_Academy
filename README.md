## حسابات التجربة المؤقتة

**ملاحظة للمشرف:**

هذه الحسابات **وهمية ومؤقتة**، وتم إنشاؤها فقط لتجربة النظام واختبار الواجهات والصلاحيات خلال مرحلة تطوير الواجهة الأمامية (Frontend)، وذلك قبل ربط النظام بقاعدة البيانات.

سيتم حذف هذه الحسابات واستبدالها بنظام تسجيل دخول وربط آمن بقاعدة البيانات في المراحل القادمة من المشروع.

### حساب المسؤول (Admin)

* **اسم المستخدم:** `admin`
* **كلمة المرور:** `admin123`

### حساب المستخدم (User)

* **اسم المستخدم:** `user`
* **كلمة المرور:** `user123`
