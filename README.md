## حسابات التجربة المؤقتة

ملاحظة للمشرف: هذه الحسابات وهمية ومخصصة فقط لمعاينة وتجربة النظام خلال مرحلة تطوير الواجهات الأمامية، وسيتم استبدالها بنظام أمان متكامل بمجرد ربط قاعدة البيانات في المراحل القادمة.

يمكنك استخدام البيانات التالية لتسجيل الدخول واختبار الصلاحيات:

* حساب المسؤول (Admin):
  * اسم المستخدم: admin
  * كلمة المرور: admin123

* حساب المستخدم (User):
  * اسم المستخدم: user
  * كلمة المرور: user123
